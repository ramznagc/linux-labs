# 🐧 Hands-on Linux 03 – Managing Users and Groups

## 📖 Overview

This hands-on lab focuses on Linux user and group management. It covers the essential commands used to create, modify, and manage user accounts, passwords, and groups in a Linux system.

---

## 📚 Topics Covered

- Basic user commands (`whoami`, `who`, `w`, `id`, `su`)
- User management (`useradd`, `userdel`, `usermod`)
- Password management (`passwd`, `/etc/shadow`)
- Group management (`groupadd`, `groupdel`, `groupmod`)
- Managing group memberships with `gpasswd`

---

## 🎯 Learning Outcomes

After completing this lab, I can:

- Understand Linux user and group concepts
- Create and manage user accounts
- Modify user information
- Manage passwords securely
- Create and manage Linux groups
- Add and remove users from groups
